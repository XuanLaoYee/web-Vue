# WebDesign
本项目是吉林大学2018级软件网页设计

####推荐使用webStorm进行编辑

###进行编写前，下载好node.js并配置好环境变量后，cmd进入终端,cd到本项目文件夹下运行
```
npm install
```
###安装好后请安装vue-cli
```
npm install -g @vue/cli
```
###安装完后，在终端cd到该项目目录下执行以下命令，进入Vue GUI界面
```
vue ui
```
###在vue的UI界面中，在左侧“依赖”这个位置，安装如下依赖
```
axios
core-js
element-ui
vue
vue-markdown
vue-router
vuex
```

### vue服务器热启动指令
```
npm run serve
```
你可以启动服务器后编写代码，这样页面可以实时热更新



### 前端项目写完后编译打包
```
npm run build
```
